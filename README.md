# Student Management System in C

##  Project Overview
This is a menu-driven **Student Management System** written in C language.  
It allows users to:
- Add new student
- Display all students
- Search by ID
- Update record
- Delete record  
- Uses **File Handling** to store data permanently.

---

##  Files Used
- **student_management.c** → Main source code
- **students.txt** → Stores student records (auto-created)
- **temp.txt** → Temporary file used for update/delete

---

##  Features Explained
### 1️⃣ Add Student
Stores:
- ID  
- Name  
- Age  
- Marks  

### 2️⃣ Display Students  
Reads all records from the file and prints them.

### 3️⃣ Search Student  
Searches a student by ID.

### 4️⃣ Delete Student  
Deletes a student record by ID using temporary file handling.

### 5️⃣ Update Student  
Updates name, age, or marks of a particular student.

---

## ▶ How to Run (Mobile)
1. Open **Cxxdroid**  
2. Create new file → `student_management.c`  
3. Paste the code  
4. Run  
5. Program will create `students.txt` automatically
